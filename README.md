# 学术塔防（academic-td）

网页可玩原型：学术主题文字合成塔防。组会招募刀/枪/弓/骑，同字合成，拼字激活武将，守护「稿」。

## 运行

```bash
cd academic-td
npm install
npm run dev
```

浏览器打开终端提示的本地地址（默认 http://localhost:5173）。

## 玩法

- **排位**：异步影子压力，结算吃本地段位星（小学→院士）
- **好友热座**：同屏上下分屏，不计星
- **招募**：耗咖啡；刷新免费 1 次；精准检索每局 1 次（指定兵种）
- **高校武将**：清华、北大、复旦、交大、浙大、南开（相邻拼字激活）

## 操作

- 拖拽招募栏单位到棋盘放置
- 拖到同级同种单位上合成
- 点击主动技按钮放武将技能
